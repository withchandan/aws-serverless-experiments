# AWS serverless experiments

## Description

A repo to run aws serverless experiments

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Deploy with Serverless

```bash
# Deploy with default config
$ sls deploy

# Remove deployed services
$ sls remove
```
